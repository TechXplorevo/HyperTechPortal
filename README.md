HyperTechPortal – Futuristic Startup Hub
Project Overview

HyperTechPortal is a futuristic, animated, and professional web hub designed for startups, AI tools, entrepreneurship, and tech innovation. It acts as a centralized platform connecting multiple projects:

Entrepreneurship Cell MES WCOE
XTN – Xplorevo Tech Network
AI Tools for Founders
Startup Radar India Blog & Grants

The website is fully mobile responsive, interactive, and visually stunning with neon glow, holographic effects, tech lines, and particle animations. It’s optimized for Netlify deployment.

Key Features
Futuristic animated UI: Tech lines, particles, holographic effects, smooth transitions.
Hero “Tech BOOM” animation: Lines converge and explode for a high-tech landing experience.
Sections for all platforms: MES WCOE, XTN, AI Tools, Blogs & Funding.
Animated neon buttons & cards: Hover glow, ripple, and particle bursts.
Typing animation: For headings and key information (using Typed.js).
Mobile responsive: Fits perfectly across mobile, tablet, and desktop screens.
Professional & energetic copy: Engaging, tech-oriented, and startup-friendly tone.
Background music support: Optional futuristic/techno music with mute/unmute toggle.
Community & social links: LinkedIn, Instagram, WhatsApp community, and email.
Ready for deployment: Fully coded HTML, CSS, and JS optimized for Netlify.

Installation & Deployment
Clone the repository:
git clone https://github.com/yourusername/hypertechportal.git
Navigate to the project folder:
cd hypertechportal
Deploy on Netlify:
Drag and drop the project folder into Netlify
Or connect your GitHub repository for automatic deploys
Optional: Add background music by replacing music.mp3 in /assets/audio/.
Customization
Links & buttons: Update href attributes in index.html.
Animations: Modify GSAP/Three.js scripts in /js/animations.js.
Theme & colors: Adjust neon colors in /css/style.css.
Fonts: Update fonts in /css/style.css using Google Fonts.
